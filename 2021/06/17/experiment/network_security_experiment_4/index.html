问题分析

1

配置php环境和安装 mairdb 数据库时，遇到一些问题：安装环境首先就是去看官方文档看怎么安装和配置。但是没有找到。网上的博客大多数都是几个环境一起配置的，如 apache , MySQL , php 和 tomecat 一起配置。没有单独配置 php。后面发现老师提供了一个多环境一同配置的软件 xampp。安装之后发现不对劲。首先，我本机就有tomecat 和 MySQL，再安装一次 MySQL 就出现了两个数据库冲突了。后面就只用了里面的 php 环境，但是并没有用里面的 MySQL 数据库。

我更加推荐以下的配置：

  1. php 官方网站下载 php
    网站：https://windows.php.net/download#php-8.0
    版本：VS16 x64 Thread Safe (2021-May-05 01:32:29)
    配置：根据应用进行配置，如果本机使用，就配置到系统环境里面。如果想只给网站使用，就给网站配置php映射。
  2. MySQL
    网站：https://downloads.mysql.com/archives/community/
    版本：8.0.24
    配置：这个可以看网上的博客
  3. 数据库可视化软件 navicat
    网站：https://navicat.com.cn/products/navicat-premium
    配置：这个可以找免安装的破解版。要安装的破解版也有。
    有了数据库和可视化软件后，就可以更加简单的使用数据库。
  4. 数据库可视化软件还可以选择 DataGrip
    网站：https://www.jetbrains.com/datagrip/
    这个更加好用一些，更加推荐这个

2

浏览器打不开 .php 文件

解决：打开 IIS 管理器，点击网站，找到“mime 类型”，添加 mime 类型。然后找到“处理程序映射”，添加模块映射。
