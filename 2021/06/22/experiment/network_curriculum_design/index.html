配置

安装 Cisco Packet Tracer

  1. 下载好安装器Cisco Packet Tracer 6.2 for Windows Student Version (no tutorials).exe
  2. 双击打开安装器，点击Next
  3. 选择I accept the agreement，然后点击Next
  4. 点击Browse，选择 Cisco 要安装的路径，点击Next
  5. 点击Next
  6. 点击Next
  7. 点击Install
  8. 安装完成后，打开软件

构建拓扑结构和显示端口

  1. 选择左下角的路由器图标，选择2811路由器，然后点击软件中央白板处，放置一个2811路由器（或者直接拖到路由器到白板中）

  2. 选择主机图片，选择Server-PT服务器，放置一个在路由器的旁边

  3. 选择闪电图标，选择Copper Cross-Over线。然后点击2811路由器，选择FastEthernet0/0端口；另一端点击Server-PT服务器，选择FastEthernet0端口。

  4. 连完线后，拓扑就完成了。显示端口：点击菜单栏Options->Preferences，勾上Always Show Port Labels。
  5. 效果
