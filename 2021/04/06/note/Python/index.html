算法

K-means

思路：

  1. 数据集
  2. 确定聚类个数 k
  3. 占位符数组
    • 数据集数据个数
    • 聚类中心数组
    • 欧式距离数组
    • 剧烈标签数组
    • 颜色数组
  4. 初始化聚类中心：随机选取k个数据点
  5. 计算E：数据到每一个聚类中心的欧式距离
    数据归入最小E的聚类中心中
  6. 计算M：根据每一个聚类的数据集，重新计算聚类中心
  7. 重复5、6

遗传算法

思路：

  1. 初始化种群
  2. 种群第一代G=0
  3. 计算适应度
  4. 选择操作
  5. 交叉操作
  6. 变异操作
  7. 产生下一代
  8. G=G+1 G>GEN
  9. 若否则回到第3步重新开始
  10. 若是则输出最佳方案
